    Section 3.1 What is a Function?

    Objectives

    In this section, we learn how to:
    🔗
    1. Articulate the definition of a function, and the connection between each input’s output.
      🔗
      🔗
    2. Identify the domain and range of a function.
      🔗
      🔗
    3. Represent functions in various ways, including tables, graphs, and equations.
      🔗
      🔗
    🔗

    Subsection Applications

    Intro:
    1. A consulting firm changes a flat fee of \(\$500\) plus \(\$50\) per hour for a project, plus \(\$350\) for each consultant who works on the project. If \(h\) represents the number of hours worked and \(c\) represents the number of consultants hired, describe a formula for the total charge for a project.
      🔗
      🔗
    2. A car rental company charges \(\$25\) per day plus \(\$0.15\) per mile driven. If \(d\) represents the number of days the car is rented and \(m\) represents the number of miles driven, describe a formula for the total charge for renting a car.
      🔗
      🔗
    🔗
    🔗

    Subsection Functions by formula

    Principle 3.1.1.

    A function is a rule that assigns to each input exactly one output.
    🔗
    🔗
    We can also place the "exactly one output" information befor the "to each input" by saying: a function is a rule that assigns exactly one output to each input.
    🔗

    Note 3.1.2.

    The important thing to emphasize is that a function will generally have many inputs, but that each of the inputs will have exactly one output. It is not possible for an input to have two outputs, and it is not possible for an input to have no output at all. We say the word rule to try to emphasize the idea that the output that we get for a specific input is determined by some kind of process or procedure that anyone can follow. For one specific function, the rule needs to be specific enough that when two people are thinking about the same input, they will both come up with the same output.
    🔗
    🔗
    There are many ways to introduce a function. One of the ways to introduce a function is to represent the function using a formula. For example, the formula \(f(x) = 3x + 1\) represents a function that takes an input \(x\) and produces an output by multiplying the input by \(3\) and then adding \(1\text{.}\)
    🔗
    In Note 1.1.2, we mentioned how important it is to pay attention to the language of mathematics. For this function, when we see \(f(x) = 3x + 1\) it is important to say (or think internally) the following: \(f\) of \(x\) equals \(3x + 1\text{.}\) The temptation exists to leave out the word "of". In earlier contexts in mathematics, including in this class, when we would have seen something like \(c(d)\) or \((c)(d)\text{,}\) we would have interpreted that writing as \(c\) multiplied by \(d\text{.}\) In those earlier contexts, that would be totally correct, and so it would lead us to the belief that we should read \(f(x)\) as if it meant \(f\) multiplied by \(x\text{.}\) But in this context, we are using the notation \(f(x)\) to mean that we are talking about a function called \(f\text{,}\) and that \(x\) represents a typical input of this function. In this context, there is no multiplication going on at all. So, this notation is admittedly inconsistent and potentially misleading, thus confusing. In the context of functions, we should make a habit of including the word "of" in our language,even if we haven’t written the word "of" when we write the notation \(f(x)\text{,}\) so that we don’t mislead ourselves.
    🔗

    Example 3.1.3.

    Given the function \(f(x) = 3x + 1\text{,}\)
    1. Find \(f(2)\)
      🔗
      🔗
    2. Find \(f(5)\)
      🔗
      🔗
    3. Find \(f(10)\)
      🔗
      🔗
    🔗
    Answer.
    1. \(f(2) = 3(2) + 1 = 6 + 1 = 7\text{.}\) In summary, \(f(2) = 7\text{,}\) which means that when the input is \(2\text{,}\) the output is \(7\text{.}\) To say \(f(2) = 7\) in words, we would say “\(f\) of \(2\) equals \(7\)”, focused on ensuring that we say the word “of”.
      🔗
      🔗
    2. \(f(5) = 3(5) + 1 = 15 + 1 = 16\text{.}\) In summary, \(f(5) = 16\text{,}\) which means that when the input is \(5\text{,}\) the output is \(16\text{.}\) To say \(f(5) = 16\) in words, we would say “\(f\) of \(5\) equals \(16\)”, focused on ensuring that we say the word “of”.
      🔗
      🔗
    3. \(f(10) = 3(10) + 1 = 30 + 1 = 31\text{.}\) In summary, \(f(10) = 31\text{,}\) which means that when the input is \(10\text{,}\) the output is \(31\text{.}\) To say \(f(10) = 31\) in words, we would say “\(f\) of \(10\) equals \(31\)”, focused on ensuring that we say the word “of”.
      🔗
      🔗
    🔗
    🔗
    🔗
    It is important to point out that the previous discussion was about one function, and not three functions. For the one function \(f(x) = 3x + 1\text{,}\) we discussed three different inputs:
    1. When the input is \(2\text{,}\) the output is \(7\text{.}\)
      🔗
      🔗
    2. When the input is \(5\text{,}\) the output is \(16\text{.}\)
      🔗
      🔗
    3. When the input is \(10\text{,}\) the output is \(31\text{.}\)
      🔗
      🔗
    The list above is meant to emphasize helpful words when discussing functions: input and output.

    Principle 3.1.4.

    When discussing functions, always think using the words input and output.
    🔗
    🔗
    The words “input” and “output” are helpful because they really describe what is going on. Functions serve as one of the most important concepts in algebra, and confusion is commen when we don’t use the words “input” and “output” to describe what’s going on. We often informally describe a function as a "machine" that takes input and produces output. This is a helpful way to think of a function. Think about the specific example of an ATM machine: you enter in your debit card and your PIN, and the machine produces gives you cash. In this analogy, the combination of your debit card and your PIN is the input, and the cash that you receive is the output. In the last example, when we wrote \(f(2)=7\text{,}\) we say that \(2\) is the input and \(7\) is the output.
    🔗
    Suppose we have the same function \(f(x) = 3x+1\) from the previous example and we are asked to find \(f(2+10)\text{.}\) Here are two responses that someone might give:
    • \(f(2+10) = f(2)+f(10) = 7 + 31 = 38\text{.}\) Here, we have borrowed our work from earlier.
      🔗
      🔗
    • \(f(2+10) = f(12) = 3(12) + 1 = 36 + 1 = 37\text{.}\) This time, we simplified \(2+10\) first.
      🔗
      🔗
    The two responses give us different final answers. In fact, one of these is incorrect. In the first response, turning \(f(2+10)\) into \(f(2)+f(10)\) is an error. That is, a function does not “distribute”.

    Warning 3.1.5. Functions do not distribute.

    A function does not distribute over addition or multiplication. That is, in general, \(f(a+b) \neq f(a) + f(b)\) and \(f(ab) \neq f(a)f(b)\text{.}\)
    🔗
    🔗
    As specific examples of this warning:
    • \(f(3+5)\) does not turn into \(f(3)+f(5)\text{.}\)
      🔗
      🔗
    • \(f(3+h)\) does not turn into \(f(3)+f(h)\text{.}\)
      🔗
      🔗
    • \(f(a+b)\) does not turn into \(f(a)+f(b)\text{.}\)
      🔗
      🔗
    • \(f(a \cdot b)\) does not turn into \(f(a) \cdot f(b)\text{.}\)
      🔗
      🔗
    🔗

    Example 3.1.6.

    Given the function \(f(x) = x^2 + 10\text{,}\) find \(f(3)\) and \(f(4)\) and \(f(-4)\text{.}\) For each of these, practice the language that should be spoken when writing in function notation. For each of these, practice the fundamental concepts of functions by writing a sentence that uses the words “input” and “output”.
    🔗
    🔗
    What we saw in the last example was that \(f(4)\) and \(f(-4)\) are both \(26\text{.}\) Each input can only have one specific output, but this example of a function shows that it is possible for two different inputs to have the same output.
    🔗

    Principle 3.1.7.

    For a given function:
    1. Each input has exactly one output. (An input may not have more than one output, and an input may not have no output.)
      🔗
      🔗
    2. It is possible for two different inputs to have the same output.
      🔗
      🔗
    🔗
    🔗
    In the next example, we compare two different functions. When there is more than one function involved, it is common to name one function \(f(x)\) and the other function \(g(x)\text{.}\) In words, we have “\(f\) of \(x\)” and “\(g\) of \(x\)”.

    Example 3.1.8.

    Consider the function \(f(x)=2^x\) and the function \(g(x)=x^2\text{,}\) and answer the questions below:
    1. Find \(f(3)\text{.}\)
      🔗
      🔗
    2. Find \(f(-3)\text{.}\)
      🔗
      🔗
    3. Find \(g(3)\text{.}\)
      🔗
      🔗
    4. Find \(g(-3)\text{.}\)
      🔗
      🔗
    🔗
    Answer.
    • \(f(3) = 2^3 = 8\text{.}\)
      🔗
      🔗
    • \(f(-3) = 2^{-3} = \frac1{2^3} = \frac{1}{8}\text{.}\)
      🔗
      🔗
    • \(g(3) = 3^2 = 9\text{.}\)
      🔗
      🔗
    • \(g(-3) = (-3)^2 = 9\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗
    In this example, we see that the function \(g\) has the property that two different inputs \(3\) and \(-3\) have the same output \(9\text{.}\) For the function \(f\text{,}\) we have only dealt with two inputs and got different outputs for each input. We may just not yet have discovered two different inputs for \(f\) that lead to the same output: more work remains to be done to figure out if \(f\) has the same phenomenon as \(g\) or not. We will discuss this specific function \(f\) in a future section, but if you’re willing to believe it for now: \(f\) is a function that has the property that two different inputs will never have the same output.
    🔗
    One vital task is taking a given function and evaluating the function for a requested input, reporting on the output. We have already done that, but we slow down intentionally right now to be absolutely sure that this vital skill is clear.

    Example 3.1.9.

    Consider the function \(f(x) = 3x^2 + 5x\text{.}\) Find \(f(6)\text{.}\)
    🔗
    Answer 1.
    \(f(6) = 3(6)^2 + 5(6) = 3(36) + 30 = 108 + 30 = 138\text{.}\)
    🔗
    🔗
    Answer 2.
    \(f(6) = 3(6)^2 + 5(6)\text{.}\)
    🔗
    🔗
    🔗
    In the second answer, we did not bother simplifying. Of course, \(3(6)^2 + 5(6)\) can be simplified to \(138\) but we intentionally did not, so that we discuss what is happening more clearly, and see a very important pattern. The first sentence “Consider the function \(f(x) = 3x^2 + 5x\)” introduced the function to us. The notation \(f(x) = 3x^2 + 5x\) in this sentence gives us the formula for a function, and the \(x\) that appears represents a generic (or typical) input. The second sentence “Find \(f(6)\)” tells us that we are now going to evaluate the function for a specific input, namely \(6\text{.}\) The specific input is \(6\text{,}\) and this specific input must replace the generic input \(x\) from the previous sentence.
    🔗
    Let’s describe what happens for this example using very specific language: Replace every \(x\) in \(f(x)=3x^2+5x\) with \(6\text{.}\)
    🔗

    Example 3.1.10.

    Consider the function \(f(x) = 3x^2 + 5x\text{.}\) Find \(f(10)\text{.}\)
    🔗
    Answer.
    \(f(10) = 3(10)^2 + 5(10)\text{.}\)
    🔗
    🔗
    🔗
    We did not simplify so that we can see the pattern better. Here’s what we did: Replace every \(x\) in \(f(x)=3x^2+5x\) with \(10\text{.}\) If we examine the previous sentence, there are three bits of notation:
    • \(\displaystyle x\)
      🔗
    • \(\displaystyle f(x)=3x^2+5x\)
      🔗
    • \(\displaystyle 10\)
      🔗
    The second bit of notation \(f(x)=3x^2+5x\) tells us what text we start with. The first notation (which is \(x\)) tells us what is going to get replaced. The third notation (which is \(10\)) tells us what we are replacing with. We need all three pieces.
    🔗

    Principle 3.1.11.

    To evaluate a function at a specific input, we need three pieces of information, but there are usually two distinct segments of notation provided:
    • The description of the function, which usually starts \(f(x) =\) but may start \(g(x)=\) or \(h(u)=\)
      🔗
      🔗
    • The evaluation request: for example, \(f(10)\text{.}\)
      🔗
      🔗
    Out of these two segments of notation, we can extract the three pieces of information that we need:
    • The starting text (which is the entire description of the function, including the \(f(x)=\) part)
      🔗
      🔗
    • The text which will have replaced, which is the variable appearing in parentheses in the description of the function.
      🔗
      🔗
    • The replacement text, which is the value inside the parentheses in the evaluation request.
      🔗
      🔗
    🔗
    🔗
    Based on this principle, let’s go back to the previous example and state various ways that we could successfully answer the question. First, we recall: Consider the function \(f(x) = 3x^2 + 5x\text{,}\) and find \(f(10)\text{.}\)
    • Replace every \(x\) in \(f(x) = 3x^2 + 5x\) with \(10\text{.}\)
      🔗
      🔗
    • Start with \(f(x) = 3x^2 + 5x\) and replace every \(x\) with \(10\text{.}\)
      🔗
      🔗
    • Imagine that \(f(x) = 3x^2 + 5x\) is typed as the only text in a Word document. In the search/replace feature, fill the “search for” box with \(x\) and fill the “replace” box with \(10\text{.}\)
      🔗
      🔗
    • Write \(f(x) = 3x^2 + 5x\text{.}\) Erase every place there is an \(x\) written. (There is one \(x\) on the left, and there are two on the right.) Every place that there is evidence of erasing, write a \(10\) instead.
      🔗
      🔗
    Finally, it is important that we emphasize that every instance of \(x\) must be replaced with \(10\text{.}\)

    Warning 3.1.12.

    When evaluating a function, every instance of the variable must be replaced with the specific input.
    🔗
    🔗
    As an exmple, writing \(f(x) = 3(10)^2+5(10)\) has the correct right side, but the left side is incorrect because it still has an \(x\text{:}\) even the \(x\) appearing to the left of the equal sign must be replaced with \(10\text{.}\)

    Example 3.1.13.

    Consider the function \(f(x) = 3x^2 + 5x\text{.}\) Find \(f(a)\text{.}\)
    🔗
    Answer.
    \(f(a) = 3(a)^2 + 5(a)\text{.}\)
    🔗
    🔗
    🔗
    We replaced every \(x\) in \(f(x) = 3x^2+5x\) with \(a\text{.}\) Said differently, we can imagine that we wrote \(f(x)=3x^2+5x\text{,}\) then erased all three spots where \(x\) appeared, and then wrote \(a\) in its place. Having to replace \(x\) with the letter \(a\) instead of a number like \(10\) sometimes causes confusion. We should end with \(f(a)=3a^2+5a\text{,}\) but here are some common incorrect answers:
    • \(f(a) = f(3a^2+5a)\) is incorrect, because of the \(f\) that appears on the right side. In the original text \(f(x)=3x^2+5x\text{,}\) there is no \(f\) on the right side, so introducing one is artificial.
      🔗
      🔗
    • \((3x^2 +5x)(a)\) is incorrect. This appears to be \(f(x) \cdot a\text{,}\) that is, \(f(x)\) multiplied by \(a\text{.}\)
      🔗
      🔗

    Example 3.1.14.

    Consider the function \(f(x) = 3x^2 + 5x\text{.}\) Find \(f(a+h)\text{.}\)
    🔗
    Answer.
    \(f(a+h) = 3(a+h)^2 + 5(a+h)\text{.}\)
    🔗
    🔗
    🔗
    This last example is one that causes a lot of struggle, because in \(f(x) = 3x^2+5x\) we are replacing each \(x\) with \(a+h\text{.}\) In that sense, this feels harder than previous examples where we replaced \(x\) with a single number or a single letter. However, even in Example 3.1.10 we could argue that replacing each \(x\) with \(10\) already involves replacing the single “character” with two characters “1” and “0”. Of course, we don’t tend to think of \(10\) as two separate symbols (and instead tend to treat \(10\) as if it’s just one “thing”), but if we can replace \(x\) with two characters by writing \(10\) each time, then we can also replace \(x\) with three characters by writing \(a+h\) each time. We mention some common incorrect responses to this question:
    • \(f(a+h) = f(3(a+h)^2 + 5(a+h))\text{.}\) This response is close, but there is no \(f\) on the right side of the original description of the function.
      🔗
      🔗
    • \(f(a+h) = f(a)+f(h) = 3a^2 + 5a + 3h^2 + 5h\text{.}\) This response is incorrect because a function does not distribute, as mentioned in Warning 3.1.5.
      🔗
      🔗
    • \(3a^2+5a+h\) is incorrect. This is the value of \(f(a)+h\text{,}\) not \(f(a+h)\text{.}\)
      🔗
      🔗
    🔗

    Example 3.1.15.

    Given \(f(x) = 10^x + x^5 + \frac{4}{x}\text{,}\) describe the process for finding \(f(3+r)\text{.}\)
    🔗
    Answer.
    Replace every \(x\) in \(f(x) = 10^x + x^5 + \frac{4}{x}\) with \(3+r\text{.}\)
    🔗
    🔗
    🔗
    The result of doing this would give: \(f(3+r) = 10^{3+r} + (3+r)^5 + \frac{4}{3+r}\text{.}\) Note that there is no \(f\) to the right of the equal sign. In our writing, we wrote \(3+r\) once before the equal sign and three times after the equal sign.
    🔗

    Example 3.1.16.

    Given \(f(x) = \sqrt{10 + x} + \frac{6}{x^2 + 1}\text{,}\) describe the process for finding \(f(a+b)\text{.}\)
    🔗
    Answer.
    Replace every \(x\) in \(f(x) = \sqrt{10 + x} + \frac{6}{x^2 + 1}\) with \(a+b\text{.}\)
    🔗
    🔗
    🔗
    Doing this would give: \(f(a+b) = \sqrt{10 + (a+b)} + \frac{6}{(a+b)^2 + 1}\text{.}\)
    🔗

    Example 3.1.17.

    Given \(f(x) = \dfrac{10+x}{x^2 + 1}\text{,}\) find \(f(c+d)\text{.}\)
    🔗
    Answer.
    Replace every \(x\) in \(f(x) = \dfrac{10+x}{x^2 + 1}\) with \(c+d\text{.}\) This gives us: \(f(c+d) = \dfrac{10+(c+d)}{(c+d)^2 + 1}\text{.}\)
    🔗
    🔗
    🔗
    🔗
    We now examine a question that is very typical in applications for functions, finding what is called the average rate of change.

    Example 3.1.18.

    Consider the function \(f(x) = 3x^2 + 5x\text{.}\) Find the average rate of change \(\dfrac{f(a+h) - f(a)}{h}\) and simplify.
    🔗
    Answer.
    In the fraction we are given, we notice \(f(a+h)\) as well as \(f(a)\text{,}\) but we found these earlier. Recall:
    • From Example 3.1.14, we had \(f(a+h) = 3(a+h)^2 + 5(a+h)\text{.}\)
      🔗
      🔗
    • From Example 3.1.13, we had \(f(a) = 3a^2 + 5a\text{.}\)
      🔗
      🔗
    We didn’t simplify \(f(a+h)\) earlier, because we were trying to notice a pattern. However, for the purposes of this work, let’s simplify: \(f(a+h) = 3(a+h)^2 + 5(a+h) = 3(a^2 + 2ah + h^2) + 5(a+h) = 3a^2 + 6ah + 3h^2 + 5a + 5h\text{.}\)
    🔗
    Now, let’s rewrite the requsted expression, and substitute in earlier work for both \(f(a+h)\) and \(f(a)\text{:}\) \(\dfrac{f(a+h) - f(a)}{h} = \dfrac{(3a^2 + 6ah + 3h^2 + 5a + 5h) - (3a^2 + 5a)}{h} = \dfrac{6ah + 3h^2 + 5h}{h} = \dfrac{h(6a + 3h + 5)}{h} = 6a + 3h + 5\text{.}\)

    Subsection Functions by graph

    Another way to describe a function is by a graph. A graph is a picture of the function, and it is drawn on the \(xy\)-plane. The \(x\)-axis represents the input, and the \(y\)-axis represents the output.
    🔗
    Like any graph in the \(xy\)-plane, the graph of a function is a collection of points. Every point in the graph records information about a combination of input and output: given a point \((a,b)\) in the graph, the \(x\)-coordinate represented here by \(a\) is the input, and the \(y\)-coordinate represented here by \(b\) is the output.

    Principle 3.1.27. Graph of a function.

    The point \((a,b)\) is in the graph of a function \(f\) if and only if \(f(a)=b\text{.}\)
    🔗
    🔗
    So, have a more concrete example, having the point \((8,9)\) in the graph of a function \(f\) means that \(f(8)=9\text{.}\) We can connect a function given by a table and a function given by a graph: each row of the table description corresponds to a point in the graph.
    🔗
    Consider the function \(f\) whose graph is shown below. Based on this graph,
    1. Find \(f(3)\text{.}\)
      🔗
      🔗
    2. Find \(f(4)\text{.}\)
      🔗
      🔗
    3. Find \(f(5)\text{.}\)
      🔗
      🔗
    🔗
    Answer.
    • \(f(3) = 8\) because the point \((3,8)\) is in the graph.
      🔗
      🔗
    • \(f(4) = 5\) because the point \((4,5)\) is in the graph.
      🔗
      🔗
    • \(f(5) = 2\) because the point \((5,2)\) is in the graph.
      🔗
      🔗
    🔗
    🔗
    This question helps us exercise Principle 3.1.27. When we are asked \(f(5)\text{,}\) we look at that number in parentheses (in this case, \(5\)) and call that an input, and we ask what output do we get for the input \(5\text{.}\) Said in the language of the graph, we look for the \(y\)-coordinate of the point in the graph which has \(x\)-coordinate \(2\text{.}\) If we can say it even more procedurally, we first need to go where \(x=2\text{,}\) so starting from the origin, we go \(2\) units to the right. Then, we are looking for a point with this \(x\)-value, so we scan up and down until we find a point. The \(y\)-coordinate of that point is the answer to the question \(f(2)\text{.}\) Because a function is a rule that assigns exactly one output to each input, it would be impossible for the graph to have two different points with the \(x\)-value \(2\text{.}\) In fact, it would be impossible for the graph to have two different points with the same \(x\)-value, because that would mean that there are two different outputs for the same input, which is not allowed in a function.

    Principle 3.1.29. Vertical Line Test.

    The graph of every function passes the Vertical Line Test: every vertical line intersects the graph of a function in at most one point.
    🔗
    🔗

    Example 3.1.30.

    Before we answer the questions on the parts, we will briefly explain the picture. The graph is made up of two line segments. One of those line segmenst is from \((1,2)\) to \((3,5)\text{.}\) Each endpoint is a filled point, indicating that the point \((1,2)\) is included in the graph and \((3,5)\) is also included in the graph. All points on the line segment between these two points are also included in the graph, but were not drawn in thick: we only drew the endpoints in thick to be able to clearly indicate that these points are included in the graph. The next line segment will show excluded points. The other line segment is from \((3,10)\) to \((9,8)\text{.}\) Each endpoint is an unfilled point, indicating that the point \((3,10)\) is not included in the graph and \((9,8)\) is also not included in the graph. We have all the points between \((3,10)\) and \((9,8)\) included in the graph, but not the endpoints. We even have poins thate are very close to the endpoints. For example, there is a point in the graph with an \(x\)-value of \(8.9\text{,}\) and the \(y\)-value of that point is slightly bigger than \(8\text{.}\)

    • \(f(1) = 2\text{.}\)

The slope of the line segment is \(\dfrac{5-2}{3-1} = \dfrac{3}{2}\text{,}\) and we can use the point-slope form of a line to write an equation for this line segment:
      \begin{equation*} y-2 = \dfrac{3}{2}(x-1). \end{equation*}
      This is technically the equation for a line (instead of a line segment), but every point that is on the line segment has to satisfy this equation. We know we need the point with \(x\)-coordinate \(2\text{,}\) so we can substitute \(2\) for \(x\) in the equation and find the correct value of \(y\text{:}\)
      \begin{equation*} y-2 = \dfrac{3}{2}(2-1) \end{equation*}
      \begin{equation*} y - 2 = \dfrac{3}{2} \cdot 1 \end{equation*}
      \begin{equation*} y - 2 = \dfrac{3}{2} \end{equation*}
      \begin{equation*} y = \dfrac{3}{2} + 2 \end{equation*}
      \begin{equation*} y = \dfrac{7}{2} \end{equation*}
