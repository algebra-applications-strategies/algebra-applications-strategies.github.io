    Section 3.3 Linear and Quadratic Functions

    Objectives

    In this section, we learn how to:
    🔗
    1. Model real-world situations using linear and quadratic functions.
      🔗
      🔗
    2. Graph linear and quadratic functions.
      🔗
      🔗
    3. Evaluate linear and quadratic functions.
      🔗
      🔗
    🔗

    Subsection Applications

    With the skills learned in this section, we will be able to answer questions such as:
    1. The value of a new car is estimated by a linear model. Three years after purchase, the car’s value is \(\$20,000\text{.}\) Eight years after purchase, the car’s value is \(\$10,000\text{.}\) What is the estimated value of the car ten years after purchase?
      🔗
      🔗
    2. A business has a profit function given by \(P(x) = -4x^2 + 80x - 100\text{,}\) where \(x\) is the number of toasters sold. How many toasters should the business sell to maximize profit, and what is that maximum profit?
      🔗
      🔗
    3. A ball is thrown into the air. Its height in feet after \(t\) seconds is given by the function \(h(t) = -16t^2 + 64t + 5\text{.}\) How long will it take for the ball to hit the ground? At what point is the ball at its maximum height, and what is that height?
      🔗
      🔗
    🔗
    🔗

    Subsection Linear functions and linear models

    A linear function is a function whose graph is a straight line. With the exception of vertical lines (which cannot be functions anyway, because the Vertical Line Test fails for vertical lines), every line can be written in the format \(y = mx + b\text{,}\) and so replacing \(y\) with \(f(x)\text{,}\) every linear function can be written in the format \(f(x) = mx + b\text{.}\)
    🔗
    When a linear function describes the behavior of a real-world situation, we say that we are looking at a linear model. The slope \(m\) of the linear function describes the rate of change of the situation. For instance, we have the first preview application above, which describes the value of a car over time. For this linear model, we will have \(x\) represent the time (more specifically, the number of years after purchase), and \(y\) or \(f(x)\) will represent the value of the car in dollars. Then the slope will describe the rate at which the car’s value is changing over time.
    🔗
    Before digging into the car example, let’s look at a warm up example dealing with a linear function.
    🔗

    Example 3.3.1.

    A linear function has a graph that goes through the points \((3,8)\) and \((6,20)\text{.}\) Find a formula for the function.
    🔗
    Answer.
    Behind the layer of language, we need to find an equation for the line going through the points \((3,8)\) and \((6,20)\text{.}\) The slope is \(m = \frac{20-8}{6-3} = \frac{12}{3} = 4\text{.}\) In point-slope form, using the first point, \(y - 8 = 4(x - 3)\text{.}\) To get a function, we need to solve for \(y\text{.}\) Adding \(8\) to both sides gives \(y = 4(x - 3) + 8\text{.}\) So, one formula for the linear function is \(f(x) = 4(x - 3) + 8\text{.}\)
    🔗
    We demonstrate that every linear funciton can be written in the format \(f(x) = mx + b\) by simplifying the formula we found. Distributing the \(4\) gives \(f(x) = 4x - 12 + 8\text{,}\) and combining like terms gives \(f(x) = 4x - 4\text{.}\) So, another formula for the linear function is \(f(x) = 4x - 4\text{.}\)
    🔗
    🔗
    🔗
    The slope \(4\) tells us a rate: that is, for every increase of \(x\) by one unit, the \(y\)-value (also called the output) of the function increases by \(4\text{.}\)
    🔗
    Let’s revisit one of the application preview questions, though to help us see the connections, we’ll make the final question that was asked the last part:

    Example 3.3.2.

    The value of a new car is estimated by a linear model. Three years after purchase, the car’s value is \(\$20,000\text{.}\) Eight years after purchase, the car’s value is \(\$10,000\text{.}\)
    1. Find a linear model for the value of the car over time.
      🔗
      🔗
    2. Identify the slope and interpret the slope in context.
      🔗
      🔗
    3. Use the linear model to estimate the value of the car ten years after purchase.
      🔗
      🔗
    🔗
    Answer.
    Because we are in the context of an application, the language “linear model” has come up, but we are really looking for a linear function. We will let \(x\) represent the number of years after purchase, and \(y\) represent the value of the car in dollars. Then we have two points: \((3, 20000)\) and \((8, 10000)\text{.}\) It means that we have the same kind of problem as the previous question, but with different numbers.
    1. Given the points \((3, 20000)\) and \((8, 10000)\text{,}\) the slope is \(m = \frac{10000 - 20000}{8 - 3} = \frac{-10000}{5} = -2000\text{.}\) Using the first point, we have \(y - 20000 = -2000(x - 3)\text{.}\) Solving for \(y\) gives \(y = -2000(x - 3) + 20000\text{,}\) so \(f(x) = -2000(x-3) + 20000\) is one formula for the linear function.
      🔗
      We can also distribute \(-2000\) and combine like terms to get \(f(x) = -2000x + 6000 + 20000 = -2000x + 26000\text{.}\) So \(f(x) = -2000x + 26000\) is another formula for the linear function.
      🔗
      🔗
    2. The slope is \(-2000\text{,}\) which means that for every increase of \(x\) by one unit the value of \(y\) decreases by \(2000\text{.}\) Now, we say the same information, but in context in light of the fact that \(x\) increasing by \(1\) means it’s one year later: every year, the value of the car decreases by \(\$2000\text{.}\)
      🔗
      🔗
    3. To estimate the value of the car ten years after purchase, we need to evaluate \(f(10)\text{.}\) Using the formula \(f(x) = -2000x + 26000\text{,}\) we have \(f(10) = -2000(10) + 26000 = -20000 + 26000 = 6000\text{.}\) So, the estimated value of the car ten years after purchase is \(\$6000\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗
    🔗
    🔗

    Subsection Quadratic functions and quadratic models

    We just saw that a linear function can always be written in the format \(f(x) = mx + b\text{,}\) or to prep for what we’ll see and use letters in alphabetical order, \(f(x) = ax + b\text{.}\) Instead of having the highest power of \(x\) be \(1\text{,}\) we can increase this by \(1\) to talk about a quadratic function. A quadratic function will have the highest power of \(x\) be \(2\text{,}\) and so a quadratic function can always be written in the format \(f(x) = ax^2 + bx + c\text{.}\) Whereas a linear function’s graph is always a straight line, a quadratic function’s graph is always a parabola.
    🔗
    The two forms of line we have focused on \(y = mx + b\) and \(y - y_1 = m(x - x_1)\) led to two general looks for a linear function: either \(f(x) = mx + b\) (alternatively written \(f(x) = ax + b\)) or \(f(x) = m(x - x_1) + y_1\) (alternatively written \(f(x) = y_1 + m(x - x_1)\)). Analogously, \(f(x) = ax^2 + bx + c\) is one of the formats we will focus on.
    • \(f(x) = ax^2 + bx + c\) is called the standard form of a quadratic function
      🔗
      🔗
    • \(f(x) = a(x - h)^2 + k\) is called the vertex form of a quadratic function.
      🔗
      🔗
    🔗
    When a quadratic function describes the behavior of a real-world situation, we say that we are looking at a quadratic model.
    🔗

    Example 3.3.3.

    Consider the quadratic function \(f(x) = (x-3)^2 + 100\) in vertex form, and write the function in standard form.
    🔗
    Answer.
    \(f(x) = (x-3)^2 + 100 = x^2 - 6x + 9 + 100 = x^2 - 6x + 109\text{.}\) So, the standard form of the function is \(f(x) = x^2 - 6x + 109\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.3.4.

    Consider the quadratic function \(f(x) = 10(x-3)^2 + 100\) in vertex form, and write the function in standard form.
    🔗
    Answer.
    \(f(x) = 10(x-3)^2 + 100 = 10(x^2 - 6x + 9) + 100 = 10x^2 - 60x + 90 + 100 = 10x^2 - 60x + 190\text{.}\) So, the standard form of the function is \(f(x) = 10x^2 - 60x + 190\text{.}\)
    🔗
    🔗
    🔗
    🔗
    Going from vertex form to standard form involves expanding the square and combining like terms. Going from standard form to vertex form involves completing the square. We were used to adding something to both sides, but now we’ll have to add and subtract a number on the same side.
    🔗

    Example 3.3.5.

    Consider the quadratic function \(f(x) = x^2 + 6x + 100\) in standard form, and write the function in vertex form.
    🔗
    Answer.
    The linear coefficient is \(6\text{,}\) so the second stick is \(3\) so the constant we need to add is \(9\text{.}\) If we had an equation, we would add \(9\) to, both sides, but we have the formula for a function, so we will add and subtract \(9\) on the same “side”. \(f(x) = x^2 + 6x + 9 - 9 + 100 = (x^2 + 6x + 9) + 91 = (x+3)^2 + 91\text{.}\) So, the vertex form of the function is \(f(x) = (x+3)^2 + 91\text{.}\)
    🔗
    🔗
    🔗
    🔗
    We can actually temporarily create an equation in \(x\) and \(y\) by replacing \(f(x)\) with \(y\text{,}\) and after completing the square and factoring, we can replace \(y\) with \(f(x)\) again. It will end up being the same mathematics (the same numbers are involved in completing the square), but it’s a little easier to process this way. Let’s revisit the same problem with this new technique.
    🔗

    Example 3.3.6.

    Consider the quadratic function \(f(x) = x^2 + 6x + 100\) in standard form, and write the function in vertex form.
    🔗
    Answer.
    Replacing \(f(x)\) with \(y\text{,}\) we have \(y = x^2 + 6x + 100\text{.}\) Subtracting \(100\) on both sides, \(y-100 = x^2 + 6x\text{.}\) The linear coefficient is \(6\text{,}\) so the second stick is \(3\) so the constant we need to add is \(9\text{.}\) Adding \(9\) to both sides, we have \(y - 100 + 9 = x^2 + 6x + 9\text{,}\) which simplifies to \(y - 91 = x^2 + 6x + 9\text{.}\) Factoring the right side gives \(y - 91 = (x + 3)^2\text{,}\) and adding \(91\) to both sides gives \(y = (x + 3)^2 + 91\text{.}\) Replacing \(y\) with \(f(x)\text{,}\) we have \(f(x) = (x + 3)^2 + 91\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.3.7.

    Consider the quadratic function \(f(x) = 10x^2 - 60x + 300\) in standard form, and write the function in vertex form.
    🔗
    Answer.
    Replacing \(f(x)\) with \(y\text{,}\) we have \(y = 10x^2 - 60x + 300\text{.}\) Subtracting \(300\) on both sides, \(y - 300 = 10x^2 - 60x\text{.}\) In the past for quadratic equations, we saw that before we consider the linear coefficient, second stick, and constant, we first need to ensure that the coefficient of \(x^2\) is \(1\text{,}\) often a "hidden" \(1\text{.}\) To make that happens is a multi-step process that starts with dividing both sides by \(10\text{.}\) As we do this, recall that we have divide the entire left side by \(10\text{,}\) and the entire right side by \(10\text{.}\)
    \begin{equation*} \frac{y - 300}{10} = \frac{10x^2 - 60x}{10}. \end{equation*}
    Each side has a fraction with subtraction in the numerator, so no cancellations will happen. We can leave the left side as is, but we make the choice to do steps on the left similarly to how we work on the right, splitting up each side as the difference of two fractions:
    \begin{equation*} \frac{y}{10} - \frac{300}{10} = \frac{10x^2}{10} - \frac{60x}{10}. \end{equation*}
    Now that minus signs are not in the numerator of a fraction (but instead between separate fractions), we can cancel in three of the four fractions:
    \begin{equation*} \frac{y}{10} - 30 = x^2 - 6x. \end{equation*}
    Now we can complete the square on the right side. The linear coefficient is \(-6\text{,}\) so the second stick is \(-3\) so the constant we need to add is \(9\text{.}\) Adding \(9\) to both sides,
    \begin{equation*} \frac{y}{10} - 30 + 9 = x^2 - 6x + 9. \end{equation*}
    Simplifying the left side and factoring on the right side,
    \begin{equation*} \frac{y}{10} - 21 = (x - 3)^2. \end{equation*}
    Now we solve for \(y\text{.}\) Adding \(21\) to both sides gives
    \begin{equation*} \frac{y}{10} = (x-3)^2 + 21 \end{equation*}
    To get \(y\) by itself, we multiply both sides by \(10\text{:}\)
    \begin{equation*} y = 10 ((x-3)^2 + 21). \end{equation*}
    While this techincally leaves \(y\) isolated, we can distribute the \(10\) to get a more standard look:
    \begin{equation*} y = 10(x-3)^2 + 210. \end{equation*}
    Replacing \(y\) with \(f(x)\text{,}\) we have \(f(x) = 10(x-3)^2 + 210\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.3.8.

    Consider the quadratic function \(f(x) = 2x^2 + 20x - 12\) in standard form, and write the function in vertex form.
    🔗
    Answer.
    Replacing \(f(x)\) with \(y\text{,}\) we have \(y = 2x^2 + 20x - 12\text{.}\) Subtracting \(-12\) on both sides, \(y + 12 = 2x^2 + 20x\text{.}\) Dividing both sides by \(2\text{,}\) we have
    \begin{equation*} \frac{y + 12}{2} = x^2 + 10x. \end{equation*}
    The linear coefficient is \(10\text{,}\) so the second stick is \(5\) so the constant we need to add is \(25\text{.}\) Adding \(25\) to both sides,
    \begin{equation*} \frac{y + 12}{2} + 25 = x^2 + 10x + 25. \end{equation*}
    Factoring the right side,
    \begin{equation*} \frac{y + 12}{2} + 25 = (x + 5)^2. \end{equation*}
    Now we solve for \(y\text{.}\) Subtracting \(25\) from both sides gives
    \begin{equation*} \frac{y + 12}{2} = (x + 5)^2 - 25 \end{equation*}
    Multiplying both sides by \(2\) gives
    \begin{equation*} y + 12 = 2((x + 5)^2 - 25) \end{equation*}
    Distributing the \(2\) gives
    \begin{equation*} y + 12 = 2(x + 5)^2 - 50 \end{equation*}
    Subtracting \(12\) from both sides gives
    \begin{equation*} y = 2(x + 5)^2 - 50 - 12 \end{equation*}
    Simplifying the right side,
    \begin{equation*} y = 2(x + 5)^2 - 62 \end{equation*}
    so
    \begin{equation*} f(x) = 2(x + 5)^2 - 62\text{.} \end{equation*}
    🔗
    🔗
    🔗
    A final answer can be left like this, but if we wish to have this look more literal in its conformance to the vertex form, we can write \(f(x) = 2(x - (-5))^2 - 62\text{,}\) in which case we more clearly see that \(a=2\) and \(h=-5\) and \(k=-62\text{.}\)
    🔗
    Now, we dig a little into the parts of the standard form \(f(x) = ax^2 + bx + c\) and the vertex form \(f(x) = a(x-h)^2 + k\text{.}\)
    • The \(a\) in both forms is the same, and it takes on the same value when convering between forms. (For example, in the last example, \(a=2\) in both \(f(x) = 2x^2 + 20x - 12\) and \(f(x) = 2(x + 5)^2 - 62\text{.}\))
      🔗
      🔗
    • If \(a > 0\text{,}\) the parabola opens upward, and if \(a < 0\text{,}\) the parabola opens downward. The figures below depict the two cases.
      🔗
      🔗
    While both parabolas were drawn in the context of \(x\)- and \(y\)-axes, all we really wanted to show here is that the parabolas open upward or downward depending on the sign of \(a\text{.}\) (The values of the other coefficients \(b\) and \(c\) in standard form or \(h\) and \(k\) in vertex form will affect the position and width of the parabola, but not its direction.)
    🔗
    There is a special point called the vertex of the parabola.
    • In the case of a parabola that opens upward (so \(a > 0\)) the vertex is the lowest point on the parabola. This is the point on the parabola that has the smallest \(y\)-value.
      🔗
      🔗
    • In the case of a parabola that opens downward (so \(a < 0\)) the vertex is the highest point on the parabola. This is the point on the parabola that has the largest \(y\)-value.
      🔗
      🔗
    🔗
    We introduce formulas for getting the vertex of a parabola in each form:

    Principle 3.3.11.

    If \(f(x) = a(x-h)^2 + k\) is a quadratic function in vertex form, then the vertex of the parabola is \((h,k)\text{.}\)
    🔗
    🔗
    The fact that the information about the vertex can be read right off of this form is why it is called vertex form.
    🔗
    We introduce a formula for getting the \(x\)-coordinate only of a vertex of a parabola in standard form:

    Principle 3.3.12.

    If \(f(x) = ax^2 + bx + c\) is a quadratic function in standard form, then the \(x\)-coordinate of the vertex of the parabola is \(x = -\frac{b}{2a}\text{.}\)
    🔗
    🔗
    Once the \(x\)-coordinate is known, the \(y\)-coordinate can be found by evaluating the function at the appropriate \(x\)-coordinate.
    🔗

    Example 3.3.13.

    Identify the vertex of the parabola given by \(f(x) = 2(x + 5)^2 - 62\text{.}\)
    🔗
    Answer.
    If it helps, we can rewrite the function as \(f(x) = 2(x - (-5))^2 - 62\text{,}\) and then we can see that \(h = -5\) and \(k = -62\text{.}\) So, the vertex is \((-5, -62)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.3.14.

    Identify the vertex of the parabola given by \(f(x) = 2x^2 + 20x - 12\text{.}\)
    🔗
    Answer.
    Because we have the function in standard form, we can use the formula \(x = -\frac{b}{2a}\) to find the \(x\)-coordinate of the vertex. Here, \(a = 2\) and \(b = 20\text{,}\) so \(x = -\frac{20}{2(2)} = -\frac{20}{4} = -5\text{.}\) To find the \(y\)-coordinate of the vertex, we evaluate the function at \(x = -5\text{:}\)
    \begin{equation*} f(-5) = 2(-5)^2 + 20(-5) - 12 = 2(25) - 100 - 12 = 50 - 100 - 12 = -62\text{.} \end{equation*}
    So, the vertex is \((-5, -62)\text{.}\)
    🔗
    🔗
    🔗
    Note that we recover the same vertex \((-5, -62)\) as in the previous example. This should be expected because from our work in Example 3.3.8 we are looking at two different forms of the same function.
    🔗

    Example 3.3.15.

    Use the vertex and the \(x\)-intercepts and \(y\)-intercept to sketch the graph of the parabola given by \(f(x) = 2x^2 + 20x - 12\text{.}\)
    🔗
    Answer.
    We already found the vertex to be \((-5, -62)\text{.}\) It may be helpful for us to replace the \(f(x)\) with a \(y\) so we have the following equation in \(x\) and \(y\text{:}\)
    \begin{equation*} y = 2x^2 + 20x - 12. \end{equation*}
    🔗
    Recall from Principle 2.2.7 to find the \(x\)-intercepts, we need to set \(y = 0\) and solve for \(x\text{.}\)
    \begin{equation*} 0 = 2x^2 + 20x - 12 \end{equation*}
    Because this is a quadratic equation, we can use the quadratic formula to solve for \(x\text{:}\)
    \begin{equation*} x = \frac{-20 \pm \sqrt{20^2 - 4(2)(-12)}}{2(2)} = \frac{-20 \pm \sqrt{400 + 96}}{4} = \frac{-20 \pm \sqrt{496}}{4} = \frac{-20 \pm 4\sqrt{31}}{4} = -5 \pm \sqrt{31} \end{equation*}
    So, the \(x\)-intercepts are \(x = -5 + \sqrt{31} \approx 1.4\) and \(x = -5 - \sqrt{31} \approx -11.4\text{.}\)
    🔗
    Recall frohm Principle 2.2.8 to find the \(y\)-intercept, we need to set \(x = 0\) and solve for \(y\text{.}\)
    \begin{equation*} y = 2(0)^2 + 20(0) - 12 = -12 \end{equation*}
    So, the \(y\)-intercept is \(y = -12\text{.}\)
    🔗
    With the vertex, \(x\)-intercepts, and \(y\)-intercept, we can sketch the graph of the parabola. It is helpful to not worry about scale.

    Example 3.3.18.

    A business has a profit function given by \(P(x) = -4x^2 + 80x - 100\text{,}\) where \(x\) is the number of toasters sold. How many toasters should the business sell to maximize profit, and what is that maximum profit?
    🔗
    Answer.
    For concreteness, we replace \(P(x)\) with \(y\text{:}\)
    \begin{equation*} y = -4x^2 + 80x - 100 \end{equation*}
    The \(x\)-coordinate represents the number of toasters sold, while \(y\) represents the profit. Because the coefficient of \(x^2\) is negative, the drawing would be a parabola which opens downward. So, there is a point at the top (called the vertex) in which the \(y\)-coordinate is largest on the porabola. Recall that \(y\) represents the profit, so it is this point that we are interested in.
    🔗
    To find the \(x\)-coordinate of the vertex, we use the formula \(x = -\frac{b}{2a}\) from Principle 3.3.12. Here, \(a = -4\) and \(b = 80\) and \(c=-100\text{,}\) so \(x = -\frac{80}{2(-4)} = -\frac{80}{-8} = 10\text{.}\) Now that we know the \(x\)-coordinate of the vertex, we can find the \(y\)-coordinate by evaluating the function at \(x = 10\text{:}\)
    \begin{equation*} P(10) = -4(10)^2 + 80(10) - 100 = -4(100) + 800 - 100 = -400 + 800 - 100 = 300 \end{equation*}
    So the vertex is the point \((10, 300)\text{.}\)
    🔗
    Recalling that \(x\) represents the number of toasters sold and \(y\) represents the profit, we can say that the business should sell \(10\) toasters to maximize profit, and that maximum profit is \(\$300\text{.}\)

    Example 3.3.19.

    A ball is thrown into the air. Its height in feet after \(t\) seconds is given by the function \(h(t) = -16t^2 + 64t + 5\text{.}\) How long will it take for the ball to hit the ground? At what point is the ball at its maximum height, and what is that height?
    🔗
    Answer.
    Before going too far, let us note that \(t\) plays the role of \(x\text{,}\) and represents the time. If we write \(y = -16t^2 + 64t + 5\text{,}\) then \(y\) represents the height of the ball.
    🔗
    To find when the ball hits the ground, we set \(h(t) = 0\) and solve for \(t\text{:}\)
    \begin{equation*} 0 = -16t^2 + 64t + 5 \end{equation*}
    Using the quadratic formula, we get:
    \begin{equation*} t = \frac{-64 \pm \sqrt{64^2 - 4(-16)(5)}}{2(-16)} = \frac{-64 \pm \sqrt{4096 + 320}}{-32} = \frac{-64 \pm \sqrt{4416}}{-32} \end{equation*}
    Simplifying the square root:
    \begin{equation*} t \apporx \frac{-64 \pm 66.4}{-32} \end{equation*}
    This gives us two solutions, but one of the times is negative, so we remove that possibility. The other solution is \(t \approx 4.075\) seconds.
    🔗
    To find the maximum height, we use the vertex formula \(t = -\frac{b}{2a}\text{:}\)
    \begin{equation*} t = -\frac{64}{2(-16)} = -\frac{64}{-32} = 2 \end{equation*}
    Evaluating the function at \(t = 2\text{:}\)
    \begin{equation*} h(2) = -16(2)^2 + 64(2) + 5 = -16(4) + 128 + 5 = -64 + 128 + 5 = 9 \end{equation*}
    So the vertex is the point \((2, 9)\text{.}\) Recalling that \(t\) represents the time and \(h(t)\) or \(y\)represents the height, we can say that the ball reaches its maximum height of \(9\) feet after \(2\) seconds. So the maximum height is reached at \(t = 2\) seconds, and the height is \(9\) feet.

    Subsection Summary

    1. A linear function is a function in the form \(f(x) = mx + b\) where \(m\) and \(b\) are constants, or \(f(x) = m(x-x_1) + y_1\) where \((x_1, y_1)\) is a point on the line.
      🔗
      🔗
    2. A linear model is a linear function that is used to describe a real-world situation.
      🔗
      🔗
    3. A quadratic function is a function in the form \(f(x) = ax^2 + bx + c\) where \(a\text{,}\) \(b\text{,}\) and \(c\) are constants, or \(f(x) = a(x-h)^2 + k\) where \((h,k)\) is the vertex of the parabola. In both the standard form and vertex form, the value of \(a\) is the same when converting between forms.
      🔗
      🔗
    4. To convert from vertex form to standard form, expand and collect like terms. To convert from standard form to vertex form, complete the square.
      🔗
      🔗
    5. A quadratic model is a quadratic function that is used to describe a real-world situation.
      🔗
      🔗
    6. The vertex of a parabola is the point on the parabola that is either the highest or lowest point, depending on whether the parabola opens downward or upward. (The parabola opens downward if \(a < 0\) and opens upward if \(a > 0\text{.}\)) The vertex can be read directly from the vertex form as \((h,k)\text{.}\) In standard form, the formula \(x = -\frac{b}{2a}\) gives the \(x\)-coordinate of the vertex.
      🔗
