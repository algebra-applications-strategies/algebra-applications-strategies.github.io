Example 1.6.94.

Factor \(x^3 + 5x^2 + 2x + 10\text{.}\)
🔗
Answer.
Because we are being asked to factor an expression with four terms, we should consider Strategy 1.6.93, the strategy of Factoring by Grouping. We first group the first two terms and the last two terms:
\begin{equation*} x^3 + 5x^2 + 2x + 10 = (x^3 + 5x^2) + (2x + 10). \end{equation*}
Next, we factor an expression out of each group:
\begin{equation*} (x^3 + 5x^2) + (2x + 10) = x^2(x + 5) + 2(x + 5). \end{equation*}
Now, both terms have the same binomial factor, so we can factor this binomial out:
\begin{equation*} x^2(x + 5) + 2(x + 5) = (x^2 + 2)(x + 5). \end{equation*}
Putting all of the work together, we have \(x^3 + 5x^2 + 2x + 10 = x^2(x+5) + 2(x+5) = (x^2+2)(x+5)\text{.}\)
