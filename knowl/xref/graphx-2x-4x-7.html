Example 3.4.31.

Graph the polynomial function \(f(x) = (x-2)(x-4)(x-7)\text{.}\)
🔗
Answer.
We see that \(2,4,7\) are each roots with multiplicity \(1\text{.}\) We draw a number line, and have cuts introduced at \(2,4,7\text{,}\) which generates four pieces: \((-\infty,2)\text{,}\) \((2,4)\text{,}\) \((4,7)\text{,}\) and \((7,\infty)\text{.}\) We introduce a test point in each piece, and determine the sign of the polynomial function at that test point by determining the sign of each factor. The way this table was built requires a bit of explanation. For example, in the first row, we report on the sign of \((x-2)\text{.}\) In the first interval \((-\infty,2)\text{,}\) we pick a test point, say \(x=1\text{,}\) and evaluate \((1-2)=-1\text{,}\) which is negative, so we report a \(-\) in the first column of the first row. (Note that this minus sign is not saying that \(1\) is negative, because of course \(1\) is not negative: we are saying that the vule of \((x-2)\) is negative when we plug a number in \((-\infty,2)\) for \(x\text{,}\) such as \(x=1\text{.}\))
🔗
The bottom row of the table reports on the sign of the polynomial function \((x-2)(x-4)(x-7)\) in each interval, and it is obtained by looking at the signs of the factors in each interval and multiplying them together: we are really just thinking about phrases such as “a negative times a negative is a positive” and “a negative times a positive is a negative”.
🔗
What does the bottom row of this table tell us? It says that when \(x\) is in \((-\infty,2)\text{,}\) the polynomial function is negative, so the graph of the polynomial function is below the \(x\)-axis for values of \(x\) in \((-\infty,2)\text{.}\) When \(x\) is in \((2,4)\text{,}\) the polynomial function is positive, so the graph of the polynomial function is above the \(x\)-axis for values of \(x\) in \((2,4)\text{.}\) When \(x\) is in \((4,7)\text{,}\) the polynomial function is negative, so the graph of the polynomial function is below the \(x\)-axis for values of \(x\) in \((4,7)\text{.}\) When \(x\) is in \((7,\infty)\text{,}\) the polynomial function is positive, so the graph of the polynomial function is above the \(x\)-axis for values of \(x\) in \((7,\infty)\text{.}\)
