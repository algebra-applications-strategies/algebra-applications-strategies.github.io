Example 2.2.40.

A line goes through the points \((30, 2)\) and \((45, 9)\text{.}\) State the \(y\)-coordinate of the point on the line which has \(x\)-coordinate \(100\text{.}\)
🔗
Answer.
We will make an equation for the line first, based on the points \((30, 2)\) and \((45, 9)\text{.}\) The slope is \(m = \frac{9-2}{45-30} = \frac{7}{15}\text{.}\) Using either point, we can get an equation for this line in point-slope form. Let’s choose the first point, \((30,2)\text{.}\) So, an equation for the line is \(y-2 = \frac{7}{15}(x-30)\text{.}\)
🔗
Now that we have an equation for the line, every point \((x,y)\) on the line must satisfy this equation, due to Principle 2.2.1. So, we look for the point of the form \((100,y)\) by substituting \(x=100\) into the equation for the line and getting the value of \(y\text{.}\) So we have \(y-2 = \frac{7}{15}(100-30)\text{,}\) so \(y= \frac{7}{15}(100-30)+2\text{.}\) This is a value of \(y\text{,}\) though we have not simplified the arithmetic.
