Example 2.1.10.

Suppose we have a first point \((x_1,z)\) and a second point \((x_2,z)\text{.}\) Note that \(z\) is the stated \(y\)-coordinate for both points, so the points would be drawn at the same height. Find the distance between these two points.
🔗
Answer.
We can subtract \(x\)-values to find the distance. If \(x_1\) is larger than \(x_2\text{,}\) then the distance is \(x_1-x_2\text{.}\) If \(x_2\) is larger than \(x_1\text{,}\) then the distance is \(x_2-x_1\text{.}\)
