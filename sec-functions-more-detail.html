    Section 3.2 More about functions

    Objectives

    In this section, we learn how to:
    🔗
    1. Determine the domain of a function given its formula, table, or graph.
      🔗
      🔗
    2. Determine the range of a function given its table or graph.
      🔗
      🔗
    3. Evaluate piecewise functions.
      🔗
      🔗
    🔗

    Subsection Domain and range

    Recall that a function is a rule that assigns to each input exactly one output. All of the inputs together goes by a name.

    Principle 3.2.1.

    The domain of a function is the set of all possible inputs for the function.
    🔗
    🔗
    The thing to keep in mind is that the domain is a set of inputs. So, it would not be accurate to say that the domain is “the input”. Instead, the domain is (informally) “all the inputs”. Said differently, the domain isn’t “one \(x\)”, but rather “all the \(x\)’s”. For some more clarity, a real number is in the domain if using it as input to the function produces a real number as output.

    Principle 3.2.2.

    The range of a function is the set of all possible outputs for the function.
    🔗
    🔗
    The range is also a set of outputs. In the last section, we were just introducing the concept of a function, and we encouraged you to think of each input producing an output, and to use the words “input” and “output” in your thought process. It can also be helpful to be a little more informal, and talk about “\(x\)’s” and “\(y\)’s” instead of “inputs” and “outputs”. But, no matter what, it is good to differentiate between \(x\)’s and \(y\)’s, instead of just calling eveything “it”.

    Principle 3.2.3.

    When discussing the numbers that are involved in a function, use language like “\(x\)’s” and “\(y\)’s”, or “inputs” and “outputs”.
    🔗
    🔗
    🔗
    Since the domain is the set of all possible inputs (and for clarity, the domain includes each and every real number which when used as input produces a real number as output), let’s think about the situations that would prevent having a real number as output.
    • When we have a fraction, the denominator cannot be zero. For example, in the fraction \(\dfrac{x+5}{x-3}\text{,}\) the denominator ends up being zero when \(x=3\text{.}\) So, we don’t want \(x-3=0\text{.}\) It is fine if \(x-3 > 0\) or \(x-3 < 0\text{,}\) but we don’t want \(x-3=0\text{.}\) For example, \(x-3\) is greater than zero when \(x=4\text{,}\) and \(x-3\) is less than zero when \(x=2\text{.}\) Having \(x=4\) is okay, since the value of the fraction would become \(\dfrac{4+5}{4-3} = \dfrac{9}{1} = 9\text{,}\) which is a real number. Having \(x=2\) is also okay, since the value of the fraction would become \(\dfrac{2+5}{2-3} = \dfrac{7}{-1} = -7\text{,}\) which is also a real number. So, we just don’t want the denominator to equal zero.
      🔗
      We should note that it is fine if the numerator is equal to zero. For example, in the fraction \(\dfrac{x+5}{x-3}\text{,}\) the numerator is equal to zero when \(x=-5\text{.}\) In that case, the fraction’s value is \(\dfrac{-5+5}{-5-3} = \dfrac{0}{-8} = 0\text{,}\) which is a real number.
      🔗
      🔗
    • When we have a square root, the content inside the square root cannot be negative. In other words, the content inside the square root must be greater than or equal to zero. For example, in the square root \(\sqrt{x-3}\text{,}\) the content inside the square root is \(x-3\text{.}\) So, we want \(x-3 \geq 0\text{.}\) It is a problem if \(x - 3 < 0\text{.}\) For example, if \(x=2\text{,}\) then \(x-3 = 2-3 = -1\text{,}\) which is negative, and the square root would be \(\sqrt{x-3}=\sqrt{2-3}=\sqrt{-1}\) which is not a real number. Note that square rooting zero is okay: \(\sqrt{0}=0\) because \(0^2=0\text{.}\)
      🔗
      🔗
    This gives us a procedure that we can always following when finding the domain of a function given its formula.

    Principle 3.2.4. Finding domain given a function by formula.

    Given a function by formula, to find its domain, find all fractions and square roots. By default, the domain of a function is a real number, with the following situations creating exceptions
    • We don’t want the denominator to equal zero.
      🔗
      🔗
    • We want the content inside a square root to be greater than or equal to zero.
      🔗
      🔗
    🔗
    🔗
    🔗

    Example 3.2.5.

    Find the domain of the function \(f(x) = \dfrac{x-3}{x-10}\text{.}\)
    🔗
    Answer.
    We don’t want the denominator to equal zero, so we don’t want \(x-10=0\text{.}\) Adding \(10\) to both sides, we don’t want \(x=10\text{.}\) Since \(x=10\) is a number that we don’t want, the domain is \((-\infty,10)\cup(10,\infty)\text{.}\)
    🔗
    🔗
    🔗
    Note that \(x=3\) causes the numerator to equal zero, but that is okay, and has no effect on the domain. Note that \(10\) is the only real number that is not in the domain, and we can see that in the next question (which is about the same fnuction), the only time that we don’t get a real number as output is when we try to compute \(f(10)\text{.}\)

    Example 3.2.6.

    Given the function \(f(x) = \dfrac{x-3}{x-10}\text{,}\) answer the following questions:
    1. Find \(f(20)\text{?}\)
      🔗
      🔗
    2. Find \(f(7)\text{?}\)
      🔗
      🔗
    3. Find \(f(3)\text{?}\)
      🔗
      🔗
    4. Find \(f(10)\text{?}\)
      🔗
      🔗
    🔗
    Answer.
    1. \(f(20) = \dfrac{20-3}{20-10} = \dfrac{17}{10}\text{.}\)
      🔗
      🔗
    2. \(f(7) = \dfrac{7-3}{7-10} = \dfrac{4}{-3} = -\dfrac{4}{3}\text{.}\)
      🔗
      🔗
    3. \(f(3) = \dfrac{3-3}{3-10} = \dfrac{0}{-7} = 0\text{,}\) which is okay to have zero in the numerator and a non-zero in the denominator.
      🔗
      🔗
    4. \(f(10) = \dfrac{10-3}{10-10} = \dfrac{7}{0}\text{,}\) because of the zero in the numerator.
      🔗
      🔗
    🔗
    🔗
    🔗
    🔗

    Example 3.2.7.

    Given the function \(f(x) = \dfrac{x-4}{x^2-4}\text{,}\) find the domain.
    🔗
    Answer.
    We don’t want the denominator to equal zero, so we don’t want \(x^2-4=0\text{.}\) Factoring, we don’t want \((x+2)(x-2)=0\text{.}\) So, we don’t want \(x+2=0 \text{ or } x-2=0\text{.}\) So we don’t want \(x=-2 \text{ or } x=2\text{.}\) Since \(x=-2\) and \(x=2\) are numbers that we don’t want, the domain is \((-\infty,-2)\cup(-2,2)\cup(2,\infty)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.2.8.

    Given the function \(f(x) = \sqrt{x-4}\text{,}\) find the domain.
    🔗
    Answer.
    We want the content inside the square root to be greater than or equal to zero. It is not technically correct to write \(\sqrt{x-4} \geq 0\text{.}\) Instead, we should state that we want the content inside the radical to be greater than or equal to zero. The radical tells us where to look, but we build one side of the inequality to have the content that is housed inside the radical:
    \begin{equation*} x-4 \geq 0 \end{equation*}
    Adding \(4\) to both sides, \(x \geq 4\text{.}\) The domain in interval notation is \([4,\infty)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.2.9.

    Given the function \(f(x) = \sqrt{2x+10} + \sqrt{22-x}\text{,}\) find the domain.
    🔗
    Answer.
    We want \(2x+10 \geq 0\) and want \(22-x \geq 0\text{.}\) The first inequality becomes \(x \geq -5\) and the second inequality becomes \(x \leq 22\text{.}\) So, the domain is \([-5,22]\text{.}\)
    🔗
    🔗
    🔗
    Because \(-3\) is in the interval \([-5,22]\text{,}\) we can compute \(f(-3)\) and get a real number as output: \(f(-3) = \sqrt{2(-3)+10} + \sqrt{22-(-3)} = \sqrt{4} + \sqrt{25} = 2 + 5 = 7\text{.}\) Inputting \(x=-3\) gave us the real numbmer \(7\) as an output. In summary, \(f(-3) = 7\text{.}\) Note, it is fine that \(x=-3\) is a negative number. We never square rooted a negative number. By substituting \(x=-3\text{,}\) the content inside the first square root simplified to \(4\text{,}\) and the content inside the second square root simplified to \(25\text{.}\)
    🔗

    Example 3.2.10.

    Given the function \(f(x) = \dfrac{\sqrt{2x+30}}{x^2-9}\text{,}\) find the domain.
    🔗
    Answer.
    • We want \(2x+30 \geq 0\text{,}\) so we want \(x \geq -15\text{.}\)
      🔗
      🔗
    • We don’t want \(x^2-9=0\text{.}\) Factoring, we don’t want \((x+3)(x-3)=0\text{.}\) So, we don’t want \(x+3=0 \text{ or } x-3=0\text{.}\) So we don’t want \(x=-3 \text{ or } x=3\text{.}\)
      🔗
      🔗
    Placing all our wants and don’t wants on a number line, we have Based on the picture, the domain is \([-15,-3)\cup(-3,3)\cup(3,\infty)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.2.12.

    Given the function \(f(x) = \dfrac{\sqrt{2x+4}}{x^2-9}\text{,}\) find the domain.
    🔗
    Answer.
    • We want \(2x+4 \geq 0\text{,}\) so we want \(x \geq -2\text{.}\)
      🔗
      🔗
    • We don’t want \(x^2-9=0\text{.}\) Factoring, we don’t want \((x+3)(x-3)=0\text{.}\) So, we don’t want \(x+3=0 \text{ or } x-3=0\text{.}\) So we don’t want \(x=-3 \text{ or } x=3\text{.}\)
      🔗
      🔗
    Placing all our wants and don’t wants on a number line, we have We drew an unfilled point at \(x=-3\) to indicate that we don’t want that number, but the restriction that \(x \geq -2\) already takes care of this. Based on the picture, the domain is \([-2,3)\cup(3,\infty)\text{.}\)
    🔗
    🔗
    🔗
    The last example goes to show that there is no mindless approach to the domain. Carefully process the information about what we want and what we don’t want. As a reminder, by default take all reals for the domain, but exceptions are created because we want the contents inside each square root to be greater than or equal to zero, and we don’t want the denominator of any fraction to equal zero.
    🔗

    Example 3.2.14.

    Consider the function described by the table: Find the domain and range.
    🔗
    Answer.
    The domain is the set of all inputs, and consists of \(1,2,3,4,5\text{.}\) The range is the set of all outputs, and consists of \(-1,3,0\text{.}\)
    🔗
    🔗
    🔗
    🔗
    When a function is described by a graph, the domain is the set of all \(x\)’s that are on the graph, and the range is the set of all \(y\)’s that are on the graph. To find the domain, we scan the picture from left to right and report on all present \(x\)-values. To find the range, we scan the picture from bottom to top and report on all present \(y\)-values.
    🔗

    Example 3.2.16.

    Find the domain and range of the function whose graph is shown below:

    Answer.
    The domain is the set of all \(x\)’s that are on the graph, which is \([1,3)\cup(3,9)\text{.}\) Because \(x=3\) is not included in the first interval but \(x=3\) is included in the second interval, we can present the domain just as we already have, or simplify the domain by writing \([1,9)\text{.}\)] The range is the set of all \(y\)’s that are on the graph, which is \([2,5]\cup(8,10)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Example 3.2.18.

    Find the domain and range of the function whose graph is shown below:
    🔗
    Answer.
    The domain is \([1,3)\cup(3,4) \cup [6,8) \cup [8,8] \cup (8,10]\text{.}\) Since \(x=8\) is ultimately included in the 4th interval, we can simplify the domain to be \([1,3)\cup(3,4) \cup [6,10]\text{.}\) Scanning from bottom to top, the range is \([0,0] \cup [1,5) \cup (5,10)\text{.}\)

    Subsection Piecewise functions

    There are functions that are nice to give by formula, but one formula doesn’t describe the whole function. The function may be described by one formula for some inputs, and a different formula for other inputs. These functions are called piecewise functions. Keep in mind that a function is a rule, and so each input can only produce one output.
    🔗
    A piecewise function has at least two pieces, but might have three pieces, or four pieces, or more. After being presented with a piecewise function, the task ordinarily will be to evaluate the function at a given input. For example, after being shown a piecewise function, we might be asked to find \(f(6)\text{.}\) Noting that the number inside parentheses is the input, we have \(x=6\text{,}\) and then go back to the description of the piecewise function with the information that \(x=6\) to see which piece applies.
    🔗

    Example 3.2.20.

    Consider the piecewise function
    \begin{equation*} f(x) = \begin{cases} 2x+1 & \text{if } x \leq 10 \\ -x+10 & \text{if } x > 10 \end{cases} \end{equation*}
    1. Find \(f(6)\text{.}\)
      🔗
      🔗
    2. Find \(f(10)\text{.}\)
      🔗
      🔗
    3. Find \(f(14)\text{.}\)
      🔗
      🔗
    🔗
    Answer.
    1. Since \(6 \leq 10\text{,}\) we use the first piece, and have \(f(6) = 2(6)+1 = 13\text{.}\)
      🔗
      🔗
    2. Since \(10 \leq 10\text{,}\) we use the first piece, and have \(f(10) = 2(10)+1 = 21\text{.}\)
      🔗
      🔗
    3. Since \(14 > 10\text{,}\) we use the second piece, and have \(f(14) = -14+10 = -4\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗
    When \(x=6\) only the first piece applies. It would be unnecessary and incorrect to compute \(-(6)+10\text{,}\) and it would definitely be incorrect to add that to the value \(2(6)+1\text{.}\)
    🔗

    Example 3.2.21.

    Consider the piecewise function
    \begin{equation*} f(x) = \begin{cases} x^3 + 10 & \text{if } x \leq 4 \\ x^2 - x & \text{if } 4 \lt x \lt 12 \\ 10 - x^2 & \text{if } x \geq 12 \end{cases} \end{equation*}
    1. Find \(f(2)\text{.}\)
      🔗
      🔗
    2. Find \(f(8)\text{.}\)
      🔗
      🔗
    3. Find \(f(14)\text{.}\)
      🔗
      🔗
    🔗
    Answer.
    1. Since \(2 \leq 4\text{,}\) we use the first piece, and have \(f(2) = 2^3 + 10 = 18\text{.}\)
      🔗
      🔗
    2. Since \(4 \lt 8 \lt 12\text{,}\) we use the second piece, and have \(f(8) = 8^2 - 8 = 56\text{.}\)
      🔗
      🔗
    3. Since \(14 \geq 12\text{,}\) we use the third piece, and have \(f(14) = 10 - 14^2 = -176\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗
    Note, when finding \(f(2)\) we only need to compute \((2)^3+10\) based on the first piece, and we do not need to compute \((2)^2-2\) based on the second piece, nor do we need to compute \(10-(2)^2\) based on the third piece. Moreover, \(f(2) = (2)^3+10\) and \(f(2) \not= ((2)^3+10) + ((2)^2-2) + (10-(2)^2)\text{.}\)
